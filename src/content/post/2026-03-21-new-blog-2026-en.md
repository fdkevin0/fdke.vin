# What This Blog Is Built On

As the [Cloudflare buy the Astro](https://blog.cloudflare.com/astro-joins-cloudflare/), it will be the best time build Astro site on Cloudflare Workers. Say No to Next.js. This blog currently deploy on Cloudflare Workers with Durable Objects, Queues, AI, D1 and some other features.

- Reading: RSS Feed, fetch every hour
- Photos: Publish my photos, build with [PicImpact](https://github.com/besscroft/PicImpact)
- Tools: Photograph Utils and site management
- Posts: manage on Git
