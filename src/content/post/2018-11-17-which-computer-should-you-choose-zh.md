# 现状

##### Mac OS X

现在家中使用的主要是一台iMac，配置不高，但取决于其优秀的OS X系统，使用体验相当好。当然，还有两台window的电脑，不过很少使用罢了。

iMac的入门很简单，不过要是从windows切换过来还是有些不适应。不过相比于之前使用的Linux来说，还是相当简单的。OS X很漂亮，漂亮到不敢想象。对比之下，win就是个渣。

iMac上自带了一块机械硬盘，不过使用体验极差。于是用一块三星850 500g 外接做系统盘，之后有时间会拆了换一块NVMe ssd。不过电脑只是拿来作为下载用，速度也够了。

之前一直使用了十来年的Windows，对于复杂的中国网络环境和其盗版横行的软件市场已经见怪不怪，对于各类安全管家也可以说是了如指掌。忽然换到了OS X 平台后一度不适应，包括操作系统和简洁干净的一切，那一瞬间可以说是涕泗横流了。之后则慢慢的熟悉和喜欢上了这个系统，对于自己早期的双系统的行为也表示一定程度的鄙视。

但是mac依旧有些要吐槽的:

1. 太过操蛋的键盘，键程太小，反馈不足，让人手指难受。毕竟是经常要码字的呀。但~~因为穷~~为了美观，坚持使用了自带键盘。
2. 卸载应用真的操蛋，删完了程序不管你还要不要，应用数据保证是留着的。导致系统莫名其妙占了两百多个G，只能隔个几年就重装系统。

##### Linux

后来为了追求抛弃win的追求，我又给我的笔记本换上了Deepin。这是一个相当不错的linux发行版，尤其是其界面被我当作是最美Linux，没有之一。为国人的使用优化也很到位。但是对于显卡要求有点高，我的赛扬1.6\*2表示吃不消。~~据说现在被人权势欺压已经快要倒了？~~

##### Windows10

对于windows，我只能说再见了，它有着最奇怪的的字符处理系统，经常乱码。他的安全让人担忧，我甚至可以一步步教会我的奶奶获得系统最高权限。bug太多，每次更新系统都是一堆bug，又一次更新开发者预览版甚至让我直接更新到蓝屏系统损坏，无奈重装了系统。如果对我说游戏？不好意思，鄙人不才，不喜游戏。
